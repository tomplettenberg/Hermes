# YouTube to Social Repurposer – Validation Demo

This is a public demo for validating the concept of automated YouTube transcript repurposing.  
The demo uses **mock transcript data** – no real YouTube API calls are made.

## Purpose
- Demonstrate core workflow: transcript → multiple social‑media posts
- Validate interest before building full product
- Measure engagement via simple click‑tracking

## Live Demo
[https://[USERNAME].github.io/[REPO]/](https://[USERNAME].github.io/[REPO]/)

## Features
- **Mock transcript** from a sample YouTube video
- **One‑click generation** of Twitter threads, LinkedIn posts, blog outlines
- **Counter.dev analytics** for page‑views and CTA‑clicks
- No login, no password, no API keys required

## Technical Notes
- Static HTML/CSS/JavaScript
- Hosted on GitHub Pages
- No backend, no database
- Tracking: client‑side only via [counter.dev](https://counter.dev)
